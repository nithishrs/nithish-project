# AI Instagram Content Scheduler

This project uses Google Antigravity (ADK) agents to:
- Generate Instagram content
- Validate captions and posting time
- Create a content posting schedule

## Features
- AI-generated captions & hashtags
- Best posting time suggestions
- 7-day content scheduler

## Future Enhancements
- Instagram API integration
- CSV / Google Sheets export
- Reel & Story generation
